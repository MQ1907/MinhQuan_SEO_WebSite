## Bắt đầu

Đầu tiên, hãy khởi chạy server:

```bash
npm run dev
# or
pnpm dev
```

Mở [http://localhost:3000](http://localhost:3000) with your browser to see the result.

trên trình duyệt để xem kết quả.

Bạn có thể bắt đầu chỉnh sửa bằng cách thay đổi file `app/page.tsx`. Trang web sẽ tự động cập nhật mỗi khi bạn lưu thay đổi.
